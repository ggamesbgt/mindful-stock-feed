# Mindful Stock Feed

Prototype mobile web untuk pengalaman mindful scrolling berbasis konten saham.

## Cara pakai di GitHub Pages

1. Upload `index.html`, `style.css`, dan `script.js` ke repository GitHub.
2. Buka Settings → Pages.
3. Pilih branch `main` dan folder `/root`.
4. Buka URL GitHub Pages dari browser mobile.

## Isi Prototype

- 20 konten dummy saham.
- Komposisi konten ringan, sedang, berat.
- Card mobile full-screen style.
- Tombol Simpan dan Bahas.
- Disclaimer non-rekomendasi investasi.
